# CustomKeyEcho 使用说明

## 环境搭建

```bash
# 创建 Conda 环境
conda create -n CustomKeyEcho python=3.12 --yes

# 安装依赖
conda run -n CustomKeyEcho pip install PySide6 pynput platformdirs
```

> 如遇 conda SSL 问题，使用清华镜像源：
> ```bash
> conda create -n CustomKeyEcho python=3.12 --yes --override-channels -c https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
> ```

## 启动

```bash
eval "$(conda shell.bash hook 2>/dev/null)" && conda activate CustomKeyEcho && python Main.py
```

## 使用步骤

### 1. 通用设置

位于界面顶部：

- **目标窗口**：从下拉列表选择游戏窗口，点击"刷新"更新列表。选择"(不检测焦点)"则不做窗口限制。
- **快捷键**：点击按钮后按下想要的触发键/鼠标按键。支持组合键（如 Ctrl+F6、鼠标侧键）。快捷键仅在目标窗口焦点时响应。
- **组合键间隔**：设定组合键内修饰键之间的随机延迟区间（默认 10~30ms），模拟人手按键节奏。
- **启动后自动隐藏到托盘**：勾选后下次启动直接进入后台。

### 2. 按键序列列表

- 点击"添加序列"创建新行
- 各行**独立并行执行**，互不影响
- 支持拖拽排序
- 选中后可在下方详情区编辑

### 3. 序列详情

选中一行序列后：

- **录制按键**：点击后按下键盘/鼠标按键录入序列，支持组合键（Shift+A、Ctrl+鼠标左键等），按**回车键**结束录制
- **删除/清空**：管理序列中的按键
- **间隔**：设定该序列内按键之间的随机延迟区间

支持的输入：
- 键盘：任意按键 + 组合键
- 鼠标：左键、右键、中键、侧键 1/2

### 4. 开始/停止

按下快捷键切换运行状态：

- 首次按下 → 所有序列并行启动循环
- 再次按下 → 全部停止
- 序列执行完一轮后自动从头循环

### 焦点检测

设置了目标窗口后：
- 窗口在前台 → 正常发送按键
- 窗口失去焦点 → 自动暂停
- 窗口恢复焦点 → 自动继续
- 窗口关闭 → 自动停止所有序列

### 系统托盘

- 点击关闭按钮 → 隐藏到系统托盘（程序继续运行）
- 点击最小化按钮 → 最小化窗口
- 双击托盘图标 → 恢复窗口
- 右键托盘图标 → 显示窗口 / 退出

## 配置文件

配置自动保存在 `%LOCALAPPDATA%\CustomKeyEcho\Config.json`，包含所有序列、延迟、快捷键、目标窗口等设置。下次启动自动恢复，无需重新配置。

## 常见问题

### 按键在游戏中无效

- 尝试以管理员权限运行程序
- 部分游戏使用内核级反作弊，SendInput 可能被拦截

### 无法录制回车键

回车键用于结束录制，无法作为按键序列的一部分录入。

### 修改间隔不生效

修改间隔后需要停止并重新启动回放才能生效。

### 开机自动运行

1. 勾选"启动后自动隐藏到托盘"
2. 将程序快捷方式添加到 `shell:startup`（开机启动文件夹）
3. 开机后程序自动在后台运行，游戏打开后按快捷键即可控制
