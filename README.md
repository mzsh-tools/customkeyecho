# CustomKeyEcho

自定义按键回声 — 为了打游戏方便写的自定义按键回声，用来代替鼠标宏。

## 核心功能

- 多按键序列独立并行循环回放
- 支持键盘/鼠标按键及组合键（Ctrl+A、Shift+鼠标侧键等）
- 可调随机延迟区间，防检测
- 游戏窗口焦点检测，焦点丢失自动暂停，窗口关闭自动停止
- 快捷键一键切换开始/停止（支持鼠标按键/组合键作为快捷键）
- 系统托盘后台运行，关闭窗口自动隐藏
- 配置自动持久化，重启无需重新设置

## 使用场景

游戏中需要重复按键操作的场景，如自动施法循环、连续采集等。设定按键序列后一键启动，自动循环执行。

## 注意事项

- 仅支持 Windows
- 需要以管理员权限运行（SendInput 在部分游戏中需要提权）
- 使用前请确认目标游戏允许宏操作，避免违反游戏规则
